# Customer Churn Analysis Dashboard

## Overview
This project analyzes customer churn patterns using Power BI and SQL to identify factors contributing to customer attrition and support retention strategies.

## Objectives
- Analyze churn trends
- Identify high-risk customer segments
- Measure retention KPIs
- Support data-driven decision making

## Tools Used
- Power BI
- SQL
- Excel

## Key Metrics
- Churn Rate
- Retention Rate
- Customer Lifetime Value
- Revenue Impact

## Dashboard Features
- Interactive filters
- Customer segmentation
- Churn trend analysis
- KPI monitoring

## Key Insights
- Customers with shorter tenure showed higher churn probability.
- Certain subscription plans experienced significantly higher churn rates.
- Revenue impact was concentrated among specific customer segments.
